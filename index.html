📋 Cara Pakai Bot TikTok DM
1️⃣
Install Ekstensi
Install EditThisCookie atau Cookie-Editor di Chrome.
Login ke TikTok.com → klik ekstensi → Export → simpan sebagai cookies.json
2️⃣
Export Daftar KOL
Klik tombol di bawah untuk export KOL yang belum dihubungi. Simpan CSV ke folder bot/
3️⃣
Edit config.json
Buka file bot/config.json → isi brand, produk, komisi, dan template pesan sesuai kebutuhan.
4️⃣
Jalankan Bot
Double-click bot/run.bat → pilih file CSV → bot otomatis kirim DM satu per satu.
5️⃣
Import Hasil
Setelah bot selesai, upload file bot/log/results.csv di bawah → status KOL otomatis terupdate.

💡 Tips & Batas Aman
✓ Rekomendasi
Maksimal 30 DM/sesi
Jeda 30-60 detik antar DM
Maksimal 2 sesi per hari
Gunakan akun yang aktif
✗ Hindari
Jangan kirim 100+ DM/hari
Jangan kirim pesan sama persis
Jangan jalan 24 jam nonstop
Jangan pakai akun baru
⚠ Perlu Tahu
Tombol DM hanya muncul jika kreator mengizinkan pesan
Cookies expired ~2-4 minggu, perlu export ulang
